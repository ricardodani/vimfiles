vimfiles
========

These are the plugins and the vimrc file that I'm using. Feel free to fork it
and use the way you want.

Instalation
-----------

    $ cd ~
    $ git clone https://github.com/ricardodani/vimfiles.git
    $ git submodule init
    $ git submodule update
    $ ln -s vimfiles .vim
    $ ln -s vimfiles/vimrc .vimrc
